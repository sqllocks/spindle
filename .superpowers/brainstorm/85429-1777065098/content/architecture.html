Spindle Demo Engine — Architecture

New sqllocks_spindle/demo/ module. Three modes, three output surfaces, one orchestrator.

Module Structure

sqllocks_spindle/demo/
  __init__.py            # exports: SpindleDemo, Scenario, DemoResult
  catalog.py             # ScenarioCatalog — register + discover scenarios
  orchestrator.py        # DemoOrchestrator — multi-step run + rollback
  manifest.py            # DemoManifest — tracks all created artifacts
  cleanup.py             # CleanupEngine — remove everything from a session
  estimator.py           # CostEstimator — CU impact before run
  notebook_gen.py        # NotebookGenerator — .ipynb from scenario def
  modes/
    inference.py          # InferenceDemoMode — learn → generate → compare
    streaming.py          # StreamingDemoMode — seed → stream → live PBI
    seeding.py            # SeedingDemoMode — instant multi-target env
  output/
    terminal.py           # FidelityReport — rich terminal table + pass/fail
    charts.py             # ChartRenderer — plotly inline for Jupyter
    powerbi.py            # PBIReportGenerator — .pbix + semantic model writer
  scenarios/
    retail_inference.py   # Phase 1 — built first
    healthcare_inference.py
    retail_streaming.py
    enterprise_seed.py

Data Flow — All Three Modes

MODE 1: Inference
📂 Input: CSV / DDL / live DB
↓ spindle learn
📊 ProfileResult (.spindle.json)
↓ spindle generate
🗃 Synthetic dataset
↓ spindle compare
📈 FidelityReport
↓ output surface
🖥 Terminal / 📓 Notebook charts / 📊 PBI
MODE 2: Streaming
🎯 Target: Fabric workspace
↓ seed phase
🗃 Lakehouse + Warehouse + SQL DB
↓ DemoManifest recorded
📡 Stream phase (live)
↓ Eventhouse + Event Hub
📊 PBI report animates live
↓ Ctrl+C → cleanup prompt
🧹 CleanupEngine on confirm
MODE 3: Instant Seed
🎯 Target: Fabric workspace
↓ CostEstimator runs first
💡 "~45 CUs for ~3 min"
↓ confirm → MultiWriter
🗃 All 4 targets in parallel
↓ validation gates run
✅ DemoManifest saved
↓ notebook auto-generated
📓 Ready-to-share .ipynb

CLI Commands

spindle demo list                         # show all scenarios + modes
spindle demo run retail-inference       # run in all output modes
spindle demo run retail-streaming --workspace myws
spindle demo run enterprise-seed --estimate # cost estimate only
spindle demo cleanup SESSION_ID         # remove all artifacts
spindle demo status SESSION_ID
spindle demo notebook retail-inference   # generate .ipynb only
